Google Fusion Table

Google Fusion tables can store very large structured data sets that can be accessed by the public. Please read the following information provided by google to familiarize yourself with Google Fusion Tables.

  • About Fusion Tables
  • Getting Started
  • Fusion Tables API

To get the correct link for the Fusion Table, click on the "Share" button on the top right corner of the Fusion Table main screen. Copy the "Link to Share" and paste it in to the "Link" field.
Dont forget to make the Fusion Table public.
